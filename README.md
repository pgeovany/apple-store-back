#### **POST** - Sign-up

In order to sign-up, make a post request to: https://apple-store0.herokuapp.com/sign-up \
sending a body in the format:

```
{
  name: "...",
  email: "...",
  password: "..."
}
```

#### **POST** - Sign-in

In order to sign-in, make a post request to: https://apple-store0.herokuapp.com/sign-in
sending a body in the format:

```
{
  email: "...",
  password: "..."
}
```
